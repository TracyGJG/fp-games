# Tetris

The game Ttris implemented with functional style in JavaScript without libraries. Interfaces for terminals and (soon) for the web browser.

# Usage

```bash
git clone git@github.com:chrokh/fp-games.git
cd 002-tetris
node cli
```

- Use arrow keys, wasd, or hjkl to control the pieces.
- Make as many complete rows (no gaps) as you can.
- Avoid filling the frame.
- Escape to quit (in terminal).
